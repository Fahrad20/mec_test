# Mecenate Feed App

Мобильное приложение для платформы Mecenate. Содержит ленту постов с бесконечной подгрузкой, поддержкой тем и локализацией.

## Функционал

- ✅ **Загрузка ленты**: Работает с `cursor`-пагинацией через React Query.
- ✅ **Темная и Светлая темы**: Полная поддержка двух тем с адаптивными токенами.
- ✅ **State Management**: MobX (без классов, на базе `makeAutoObservable` объектов) для управления лайками и темой.
- ✅ **Пасхалки в приложении**:
  - **Смена языка (RU <-> EN)**: 2 нажатия + удержание 3 сек по секретной области (верхний левый угол).
  - **Смена темы (Light <-> Dark)**: 3 нажатия + удержание 3 сек по секретной области.
- ✅ **UX/UI**: 
  - Bounce анимация для лайков через Reanimated.
  - Оптимистичные обновления состояния лайков.
  - Кастомные скелетоны для загрузки.
- ✅ **Линтинг и Форматирование**: Настроены ESLint + Prettier.

## Стек технологий

- **React Native / Expo**
- **MobX** (State Management)
- **React Query** (Server State)
- **i18next** (Localization)
- **React Native Reanimated** (Animations)

## Установка

1. Склонируйте конфигурацию и установите NPM модули:
   ```bash
   npm install
   ```

2. Настройка среды:
   ```bash
   cp .env.example .env
   ```

3. Запуск:
   ```bash
   npm start
   ```

## Линтинг и Форматирование

В проекте настроены ESLint и Prettier:
```bash
npm run format
npm run lint
```
