# Mecenate Feed App

Мобильное приложение для платформы Mecenate. Содержит ленту постов с подгрузкой.

## Функционал

- ✅ Загрузка ленты с `cursor`-пагинацией 
- ✅ Локальное кэширование и optimistic UI лайков 
- ✅ Смена языка "Пасхалка": двойной клик и удержание 3 сек по экрану (RU <-> EN)
- ✅ Приятные анимации: Bounce эффект для лайка, FadeIn для постов 
- ✅ Настроен Prettier + ESLint

## Установка

1. Склонируйте конфигурацию и установите NPM модули:
   ```bash
   npm install
   ```

2. Настройка среды:
   ```bash
   cp .env.example .env
   ```

3. Запуск:
   ```bash
   npm start
   ```

## Линтинг и Форматирование

В проекте настроены ESLint и Prettier:
```bash
npm run format
npm run lint
```
